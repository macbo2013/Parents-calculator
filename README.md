# Parents-calculator(亲友计算器)
一款强大的亲友计算器，可以帮助您应对各种一位叫出亲友名字设计的计算器软件。


目前可计算类型： "爸爸", "妈妈", "爷爷", "奶奶", "外公", "外婆", "哥哥", "姐姐", "弟弟", "妹妹", "姑姑", "伯伯", "叔叔", "舅舅", "姨妈", "曾祖父", "曾祖母", "外曾祖父", "外曾祖母"。
